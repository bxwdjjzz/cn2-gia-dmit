## CN2 GIA 是什么，为什么很多人非它不可

CN2 GIA（China Telecom Next Carrier Network - Global Internet Access）是中国电信旗下的高端骨干线路。跟普通 CN2 GT 的区别在于：GIA 是全程直连，去程回程都跑专属通道，不会在国内出口排大队。

普通 VPS 商家用的国际线路，一到晚上 8-11 点，国内出口拥堵，延迟从 150ms 变成 400ms，丢包动不动 10%+。CN2 GIA 的核心价值就在这里——晚高峰不掉速，延迟控制在合理范围，不会白天测试良好、晚上用得崩溃。

判断 CN2 GIA 线路的方法也简单：路由追踪里看到 `59.43.*.*` 开头的节点，那就是进了 GIA 的专属骨干。数字不骗人。
